# 1.发展阶段

原始社会是人类社会发展的最初阶段、最低阶段，分为旧石器时代和新石器时代。  

二者的区别在于人类由食物的采集者转变为生产者。  


# 2.生产力和生产关系

原始社会生产力水平低下使人们共同劳动，共同占有生产资料，在生产中结成平等互助的关系，平均分配劳动产品。  


# 3.社会组织形式

原始社会的社会组织形式是氏族。氏族是按照血缘关系结成的共同体，是原始人共同生活的基本单位。  

人们通过氏族议事会管理集体事务，一切大事都在议事会上由全氏族成年人共同讨论决定。到了原始社会末期，一些近亲的氏族构成部落，有的部落联合起来组成部落联盟  


# 1.私有制出现的原因

生产工具的改进大大推动了生产力的发展随着生产力的发展，以氏族为单位的共同劳动，被以家庭为单位的个体劳动代替，生产资料由氏族公有转归家庭私有。  

土地变成私有财产，标志着私有制的确立。私有制的产生，使少数占有较多生产资料和生活资料者成为富人，大多数氏族成员则日益陷入贫困境地  

贫富分化的加剧，使氏族成员的地位越来越不平等，原始社会解体。  


# 2.阶级和国家

阶级：在一定生产关系中处于不同地位的集团，其实质是一个群体无偿占有另一个群体的劳动。  

国家：是阶级矛盾不可调和的产物，是阶级统治的工具，统治阶级为了维护本阶级的利益，建立军队、法庭、监狱等暴力机关，这标志着国家的产生。  


# 3.奴隶社会

（1）奴隶制生产关系的特点  

在奴隶社会，奴隶主占有生产资料并完全占有奴隶；奴隶毫无人身自由，在奴隶主的强制下劳动；奴隶劳动的全部产品都归奴隶主占有和支配，奴隶只获得最低限度的生活资料。  


# （2）主要矛盾

奴隶主阶级和奴隶阶级是人类社会最早  

形成的两大对立阶级。奴隶社会的主要矛盾是奴隶阶级和奴隶主阶级之间的矛盾。  

（3）历史意义  

奴隶社会代替原始社会，促进了生产力的发展，使人类摆脱蒙昧野蛮的状态，迈人了文明时代的门槛，这是历史的进步  


# 三、封建社会

奴隶社会后期，当生产力发展到新的水平时，出现了封建制生产关系。

（1）生产力：工业革命极大推动了生产力的发展。

# 1.资本主义生产关系的特点

资本家占有一切生产资料，失去生产资料的劳动者不得不出卖自己的劳动力，受雇于资本家，成为雇佣工人，资本家在生产过程中占有工人创造的剩余价值。

资本主义剥削的实质 剩余价值：工人创造并被资本家无偿占有的那部分价值。

# 四、资本主义经济危机

资产阶级为了维持自己的统治，采取了一些缓和阶级矛盾的措施，但这些措施没有改变资本主义私有制及其剥削关系。资本主义社会的矛盾激化，带来的结果即经济危机。  


# 1.基本特征

资本主义经济危机的基本特征是生产过剩。生产过剩是相对过剩，即相对于劳动人民有支付能力的需求来说，社会生产的商品显得过剩，而不是与劳动人民的实际需求相比的绝对过剩。  

其主要表现为：大量商品卖不出去，大量生产资料被闲置，大批生产企业、银行破产，大批工人失业，生产迅速下降，信用关系被破坏，整个社会生活陷入混乱。  


# （1）直接原因

生产无限扩大的趋势与劳动人民有支付能力的需求相对缩小之间的矛盾，以及个别企业内部生产的有组织性与整个社会生产的无政府状态之间的矛盾。当矛盾尖锐化时，社会生产结构会严重失调，一方面社会生产大规模增长，另一方面工人阶级因资本家的剥削日益贫困化，从而造成社会生产严重过剩。  


# （2）根本原因

经济危机难以治愈，根本原因在于资本主义社会的基本矛盾（生产社会化和生产资料资本主义私人占有之间的矛盾）。

生产资料所有制在生产关系中起着决定性作用，是生产关系的核心，是经济制度的基础。

# 我国的生产资料所有制

我国在社会主义初级阶段坚持公有制为主体、多种所有制经济共同发展，这是中国特色社会主义制度的重要支柱，体现了中国特色社会主义制度的优越性。  


# 1.公有制经济

（1）公有制经济的构成公有制经济包括国有经济、集体经济以及混合所有制经济中的国有成分和集体成分。  

国有经济是指生产资料归国家所有的一种经济类型，包括中央和各级国家机关、事业单位和社会团体使用国有资产投资举办的企业，国家不再核发经费或核拨部分经费的事业单位等。  

集体经济属于劳动群众集体所有、实行共同劳动，在分配形式上以按劳分配为主的社会主义经济组织。在我国，集体经济是公有制经济的重要组织形式，分为农村集体经济和城镇集体经济。生产资料归集体所有。  

公有制的实现形式可以而且应该多样化一切符合社会经济发展规律的实现形式都能够而且应该使用。  


# （2）公有制经济的地位

公有制经济是社会主义经济制度的基础，在我国的基本经济制度中占主体地位  

公有制的主体地位主要体现在：公有资产在社会总资产中占优势；国有经济控制国民经济命脉，对经济发展起主导作用。国有经济起主导作用，主要体现在控制力上。对关系国民经济命脉的重要行业和关键领域，国有经济必须占支配地位。  

国有企业是国有经济最主要的实现形式是中国特色社会主义的重要物质基础和政治基础，是推进国家现代化、保障人民共同利益的重要力量，是我们党执政兴国的重要支柱和依靠力量。  


# 2.非公有制经济

（1）非公有制经济的构成  

在我国现阶段，非公有制经济包括个体经济、私营经济、港澳台投资经济、外商投资经济以及混合所有制经济中的非国有成分和非集体成分。  


# （2）非公有制经济的地位

①是社会主义市场经济的重要组成部分；  

②是稳定经济增长和改善民生的重要力量，是创业就业的主要领域，是技术创新的重要主体，是国家税收的重要来源；  

③为我国社会主义市场经济发展、政府职能转变、农村富余劳动力转移、国际市场开拓等发挥了重要作用。

多种所有制经济共同发展，有利于发挥各种所有制的长处，调动不同经济主体的积极性和创造性，有效利用各方面的资源，取长补短，激发社会主义市场经济的活力，推动经济持续健康发展

义务：完成劳动任务，提高职业技能，执行劳动安全卫生规程，遵守劳动纪律和职业道德。

# 基本经济制度决定分配制度

在社会主义初级阶段，我国坚持公有制为主体、多种所有制经济共同发展，这就决定了我国现阶段必须实行按劳分配为主体、多种分配方式并存的分配制度  


# 1.影响因素

实行按劳分配，是由一定的经济条件决定的。  

（1）生产资料公有制是实行按劳分配的前提；  

（2）社会主义公有制条件下的生产力发展水平，是实行按劳分配的物质基础；（3）社会主义条件下人们劳动的性质和特点，是实行按劳分配的直接原因。  


# 2.基本内容和要求

按劳分配是社会主义的分配原则，其基本内容和要求是：有劳动能力的社会成员必须参加劳动；在作了必要的扣除后，以劳动者提供的劳动（包括劳动数量和质量）为尺度对个人进行分配，多劳多得、少劳少得。  


# 3.意义

（1）有助于充分调动劳动者的积极性和创造性，激励劳动者努力学习科学技术、提高劳动技能，促进社会生产的发展（2）是对几千年来不劳而获的剥削制度的根本否定，体现了劳动者共同劳动、平等分配的社会地位。  


# 1.决定因素

多种所有制经济并存决定我国多种分配方式并存。  


# 2.内容

按生产要素分配，即凭借对生产要素的所有权进行分配。生产要素主要有劳动、资本、土地、知识、技术、管理、数据等。  


# 3.意义

（1）生产要素按贡献参与分配是对社会主义市场经济条件下各种生产要素所有权存在的合理性、合法性的确认；（2）有利于让一切要素的活力竞相进发，让一切创造社会财富的源泉充分涌流；（3）有利于推动资源优化配置，促进经济发展。

# 如何实现收入分配公平

1.生产决定分配，大力发展经济是根本手段。2.坚持按劳分配为主体、多种分配方式并存的制度，是制度保证  

3.要坚持居民收入增长和经济增长基本同步、劳动报酬提高和劳动生产率提高基本同步。  

4.要规范收入分配秩序，鼓励勤劳致富，保护合法收入，增加低收入者收入，扩大中等收入群体，调节过高收人，清理规范隐性收人，取缔非法收人。要履行好政府再分配调节职能，加快推进基本公共服务均等化，缩小收入分配差距。要健全以税收、社会保障、转移支付等为主要手段的再分配调节机制，强化税收调节。要重视发挥第三次分配作用，发展慈善等社会公益事业。必须坚持消除贫困。  

补充：转移支付，又称无偿支付，主要是指政府或企业无偿支付财物给其他经济主体以增加其收人，是收人再分配的一种形式。

# 考点1 市场的决定性作用


资源配置是由资源的有限性和需求的无限性的矛盾决定的。计划和市场是资源配置的两种基本手段。  


# 市场经济

市场在资源配置中起决定性作用的经济，主要通过价格、供求、竞争等机制实现  

市场局限性：  

（1）市场调节不是万能的。国防、治安、消防等公共物品的供给不能由市场来调节，枪支、弹药、爆炸物等特殊物品的制造和流通也不能由市场来调节，教育、医疗等与民生息息相关的重要服务不能完全由市场来调节，否则就不能保障国家安全、人民生命财产安全和人民生活，严重影响社会安定  

（2）市场调节具有自发性、盲目性、滞后性等病。单靠市场调节，会影响资源配置效率，导致资源浪费；会导致经济运行大起大落，社会经济不稳定；会产生不正当竞争、垄断，损害社会公平；会导致收人差距拉大。  

市场体系：经济生活中，有两类市场（商品和服务市场和生产要素市场），每类市场都是由交易双方、交易对象、交易价格等基本要素组成的。这些市场相互作用，形成有机联系的市场体系。要形成统一开放、竞争有序的市场体系。  


# 二、如何建设现代市场体系

1.建立公平开放透明的市场规则  

内容：市场准入规则、市场竞争规则和市场交易规则。  

（1）要实行统一的市场准入制度。  

（2）要实行统一的市场监管，反对地方保护，加强和改进反垄断和反不正当竞争执法。  

（3）要建立健全社会征信体系，褒扬诚信，惩戒失信。  

（4）要健全优胜劣汰市场化退出机制，从而实现市场准人畅通、市场开放有序、市场竞争充分、市场秩序规范  

2.要完善要素的市场化配置，加快完善主要由市场决定价格的机制，加快完善公平竞争的市场环境，推动资源配置依据市场规则、市场价格、市场竞争达到效益最大化和效率最优化，实现产权有效激励，要素自由流通，价格反应灵活，竞争公平有序，企业优胜劣汰的目标。  



# 原因

2.社会主义性质决定；  
3.共同富裕的目标要求

4.宏观调控的主要目标相互影响、相互制约。当经济繁荣时，经济增长速度较快，就业率提高，但可能伴随较高的通货膨胀率；当经济不景气时，虽然物价稳定，但经济增长率可能会下降，失业率会上升。为了让宏观经济运行保持在合理区间，要创新和完善宏观调控，提高政府治理效能，健全以国家发展规划为战略导向，以财政政策和货币政策为主要手段，就业、产业、投资、消费、环保、区域等政策紧密配合，目标优化、分工合理、高效协同的宏观经济治理体系。

1.社会主义市场经济是同社会主义基本制度结合在一起的，市场在国家宏观调控下对资源配置起决定性作用的经济。社会主义市场

和效益上来，在适度扩大总需求的同时，着力  

推进供给侧结构性改革。  

（3）使经济发展更多依靠内需拉动，更多依靠科技进步、劳动力素质提高、管理创新驱动。（4）建设创新引领、协同发展的产业体系，实现实体经济、科技创新、现代金融人力资源的协同发展。（5）要深化供给侧结构性改革，加快发展先进制造业，推动互联网、大数据、人工智能同实体经济深度融合，营造脚踏实地、勤劳创业、实业致富的发展环境和社会氛围。（6）建设彰显优势、协调联动的城乡区域发展体系。（7）全面实施乡村振兴战略。建立健全城乡融合发展体制机制和政策体系，加快推进农业农村现代化（8）深人实施区域协调发展战略，建立更加有效的区域协调发展新机制

# 经济全球化

商品、服务及技术、资金、劳务等要素通日益频繁的国际贸易、国际金融等在全球范围内迅速流动和广泛配置，世界经济出现了高度融合的局面。  


# 1.表现

# 2.影响因素

（1）社会生产力发展的客观要求，科技进步的必然结果。生产力的发展使得扩张国外市场的需求更加迫切，运输和通信手段的革命性变化为经济全球化奠定物质技术基础  

（2）世界各国对本国、本民族利益的追求是经济全球化加速发展的根本动因（3）市场经济体制为经济全球化奠定了体制基础。  


# 3.重要载体

跨国公司是经济全球化的重要载体。其基于利益最大化在全球范围内从事对外直接投资，在其他国家组织生产经营。跨国公司推动着生产、贸易、金融全球化的深人发展与国际分工的深化，把国际分工、国际贸易分别发展为跨国公司的内部分工和内部贸易，促进全球资源配置的优化和全球的科技合作与进步。跨国公司发展也存在着负面影响，如实行跨国行业垄断、破坏国际市场的公平竞争，向发展中国家转移落后产能与环境污染等  


# 4.机遇、挑战与对策

（1）机遇  

总体而言，经济全球化符合经济规律，符合各方利益。经济全球化促进商品、服务与生产要素的全球流动，促进国际分工水平的提高和国际贸易的迅速发展，推动资源配置效率提高、科技进步、产业转移、结构升级。有利于发展中国家充分利用国际国内两种资源、两个市场，推动自身经济发展  


# （2）挑战

在经济全球化进程中，世界经济发展面临发展不平衡、不确定性、风险加剧的问题经济全球化还会带来危机全球化  


# （3）对策

抓住机遇、积极参与、迎接挑战、合作共赢。  

①推动经济全球化朝着更加开放、包容、普惠、平衡、共赢的方向发展  

要建设创新型世界经济，开辟增长源泉要建设开放型世界经济，拓展发展空间要建设联动型世界经济，凝聚互动合力要建设包容型世界经济，夯实共赢基础②要完善全球治理，各国共同协商合作、加强合作，建立新的国际经济秩序  


# 1.现状

经过多年发展，我国全方位、宽领域、多层次的对外开放格局和开放型经济体系逐步形成，开放型经济新体制逐步健全。  


# 2.对外政策

（1）继续坚持对外开放的基本国策，坚持打开国门搞建设，更加主动参与和推动经济全球化进程，坚持实施更大范围、更宽领域、更  

深层次对外开放，依托我国超大规模市场优势，促进国际合作，实现互利共赢。  

（2）加快转变对外经济发展方式，推动开放朝着优化结构、拓展深度、提高效益方向转变，着力培育开放型经济发展新优势。  

要形成以技术、品牌、质量、服务为核心的出口竞争新优势，拓展对外贸易，推进贸易强国建设。  

（3）要实行高水平的贸易自由和投资自由化便利化政策，放宽市场准入，扩大服务业对外开放，保护外商投资合法权益  

要创新对外投资方式，促进国际产能合作，加快培育国际经济合作和竞争新优势。  


# 3.中国对世界经济的贡献

中国的发展为世界各国的发展提供了机遇。中国经济快速增长，为世界各国提供了更广阔的市场、更充足的资本、更丰富的产品、更宝贵的合作机会，为全球经济的稳定和增长提供了持续强大的推动力。中国同一大批国家的联动发展，使全球经济发展更加平衡。中国改革开放持续推进，为开放型世界经济的发展提供了重要动力  

中国奉行多边贸易体制，反对贸易保护主义，积极参与全球经济治理和规则制定，为世界经济增长发掘新动力，使世界经济增长更包容。

# 1.政党

政党是阶级斗争发展到一定阶段的产物。政党通常以掌握政权、统治国家为目标。也有政党不谋求执政，极力推广某种政治思想、维护某种政治利益。政党通常有特定的政治目标和意识形态  

在代议制政体下，政党主要通过参加选举手段争取执政。  

资本主义国家的政党制度主要有一党制、两党制、多党制  


# 2.利益集团

利益集团，是指具有共同利益，向政府或社会提供诉求，或以合法手段施加压力，借以影响公共政策，以争取其团体利益及其成员利益的社会团体。  

利益集团不以取得政权或改变政权为目标，而是在既有政治体制内进行完全合法的活动。利益集团影响政府决策的途径有游说、宣传、助选、集会游行、司法诉讼等。  

资本主义国家的利益集团，虽然在一定程度上能缓和社会矛盾，但无法从根本上解决不同阶级之间的矛盾。

# 五、党的领导

坚持党的领导，是中国革命、建设、改革事业不断取得胜利的根本政治保证。在中国特色社会主义新时代，坚持和加强党的领导，就是要确保党对一切工作的领导，确保党始终总揽全局、协调各方。  

1.政治领导，就是党在政治立场、政治方向、政治道路、政治原则方面的领导，主要体现为党的路线、方针、政策的领导。确保党和国家的事业沿着正确的方向前进，增强政治意识、大局意识、核心意识、看齐意识，自觉维护党中央权威和集中统一领导。  

2.思想领导，即党在思想理论和意识形态上的领导3.组织领导，即党在组织体系和组织工作方面的领导，就是要保证党的路线、方针、政策和重大工作部署的贯彻和落实。  

坚持党的领导、人民当家作主、依法治国有机统一是中国特色社会主义民主政治发展的必然要求。其中坚持党的领导是根本保证，人民当家作主是本质特征，依法治国是党领导人民治理国家的基本方式  


# 六、党的执政

1.科学执政，即坚持以马克思主义为指导，不断探索和遵循共产党执政规律、社会主义建设规律、人类社会发展规律，全面增强执政本领，提高长期执政能力，并按照客观规律执好政、掌好权。  

2.民主执政，即坚持为了人民执政、依靠人民执政。民主执政强调人民群众的历史主体地位。  

3.依法执政：中国共产党执政的基本方式。党要坚持依法治国，领导立法（党的意志要通过人民代表大会上升为国家意志）、保证执法、支持司法、带头守法。  

三者有机统一，其中科学执政是基本前提，民主执政是本质所在，依法执政是基本途径（也是执政的基本方式）。

人民代表大会制度是我国的根本政治制度

人民代表大会制度是按照民主集中制的原则，由人民定期选出自己的代表组成人民代表大会作为人民行使国家权力的机关，并由人民代表大会产生其他国家机关，以实现人民民主专政历史任务的政权组织形式。  


# 3.民主集中制

民主集中制即在民主基础上的集中和集中指导下的民主相结合的制度。表现在：人民代表大会与人民的关系上；人民代表大会与其他国家机关的关系上；中央与地方国家机构的关系上。

# 2.民族区域自治制度

（1）含义和地位  

民族区域自治是在国家统一领导下，各少数民族聚居的地方实行区域自治，设立自治机关，行使自治权。民族区域自治制度是我国的一项基本政治制度。  

（2）自治方式  

①级别划分 —自治区、自治州、自治县三级。②自治机关—民族自治地方的人民代表大会和人民政府  

③自治权——在行使一般行政地方国家机关职权的同时，依法行使自治权。它是自治机关根据本地方实际情况贯彻执行国家法律、政策，自主地管理本民族自治地方内部事务的权力。主要有依法制定自治条例和单行条例；依法管理地方财政的自治权；依法自主地安排和管理地方性的经济建设事业；依法自主地管理本地方的教育、科学、文化、卫生、体育事业，保护和整理民族的文化遗产，发展和繁荣民族文化。自治权是民族区域自治制度的核心。  

④前提和基础 我国的民族区域自治制度以国家统一为前提和基础，是国家的集中统一领导与民族区域自治的有机结合。  

（3）实行民族区域自治制度的原因及优势  

原因：是由我国的历史传统和现实情况决定的。我国是统一的多民族国家和“大杂居、小聚居”的民族分布特点是我国实行民族区域自治的社会基础。各民族在长期奋斗中形成相互依存的民族关系使我国的民族区域自治具有坚实的政治基础  

优势：有利于维护国家统一和安全，保障少数民族人民当家作主的权利；有利于发展我国平等、团结、互助和谐的社会主义民族关系；有利于促进我国社会主义现代化建设事业蓬勃发展。

# 1.我国法律发展的历史

（1）法治与人治  

在古代社会，特权阶层常常凌驾于法律之上；近代以来，法律面前人人平等逐渐成为现实，法治替代人治，成为一种全新的国家治理方式。  


# （2）中华法系

中华法系是指以儒家思想为理论基础，在中国古代法律的基础上发展起来的法律文明，它强调法律与伦理道德的统一。代表性法典为《唐律疏议》。  


# 2.马克思主义法治理论

（1）法的产生  

马克思主义认为，法是由社会物质生活条件决定的，是人类社会发展到一定阶段的产物。在阶级社会中，法律反映的是统治阶级的根本利益和共同意志  

（2）法的分类  

人类历史上的四种法律类型：奴隶制法、封建制法、资本主义法、社会主义法。  

（3）法的含义和作用  

法是维护社会秩序、调整社会关系的一种社会规范。在阶级社会中，法确认、保护和发展有利于统治阶级的社会关系和社会秩序。在我国，法是人民的根本利益的体现  

法是由国家制定或认可的社会规范，由国家强制力保证实施，具有普遍约束力。  

法在国家治理中既具有政治职能，又具有社会职能。  


# 3.新中国法治建设的成就

（1）形成了以宪法为核心的中国特色社会主义法律体系。  

（2）依法行政和公正司法水平不断提高。  
（3）人权的法治保障取得巨大成就。

建设中国特色社会主义法治体系、建设社会主义法治国家

# 1.内涵

科学立法就是要尊重和体现社会发展的客观规律；体现我国社会主义国家性质；符合我国的政治制度和历史传统；符合国情和实际；必须遵循法律体系的内在逻辑和立法工作规律。  


# 2.推进科学立法

（1）要做到依法立法。按照法定职权，依照法定程序开展立法工作。（2）要充分发扬民主。坚持民主立法，开展立法协商，广泛凝聚共识（3）合理设定权利与义务、权力与责任应在立法中客观地认识现实生活中的各种利益，做到权利与义务统一。  


# 4.我国有立法权的国家机关

（1）全国人民代表大会→修改宪法，制定和修改刑事、民事、国家机构的和其他的基本法律。（2）全国人大常委会→制定和修改其他法律。（3）国务院及其部门→国务院制定行政法规；国务院相关部门制定部门规章（4）设区的市（自治州）及以上的人大及其常委会→制定地方性规（5）设区的市及以上的地方政府→制定地方政府规章。（6）自治区、自治州、自治县的人大→制定自治条例和单行条例。  


# 1.内涵

严格执法，就是执法机关在执法过程中严格依法办事。  

严格执法有利于捍卫法律的权威和尊严；有助于实现社会公平正义；有助于推进建设法治政府。  


# 2.推进严格执法

（1）要全面履行政府职能，行政机关要坚持法定职责必须为、法无授权不可为。（2）坚持规范执法，严密的执法程序是规范执法的重要前提，也是公开公正执法的重要保障。（3）坚持公正执法，同等情况下平等对待，不同情况下差别对待；恰当地行使自由裁量权，不得违背法律精神和原则（4）坚持文明执法。执法部门改进执法方式，做到语言、行为规范，融法、理、情于一体。  


# 1.内涵

公正司法，就是要在司法活动的过程和结果中坚持和体现公平正义。  

（1）司法的程序要公正。意味着当事人的诉讼地位平等，司法过程严格依据诉讼法进行。通过程序公正可以最大化实现结果公正。（2）司法的结果要公正。意味着法律适用准确，案件事实清楚，裁判结果合法合理  


# 2.推进公正司法

（1）必须确保审判权和检察权依法独立行使。各级党政机关要支持法院、检察院依法独立公正行使职权。（2）必须坚持以事实为根据，以法律为准绳，做到事实认定符合客观真相、办案结果公正、程序公正。（3）必须坚持人民司法为人民，依靠人民推进公正司法、通过公正司法维护人民权益依法及时公开司法依据、程序、流程、结果和生效法律文书，杜绝暗箱操作。（4）加强公正司法，必须加强人权司法保障，强化诉讼过程中当事人和其他诉讼参与人的知情权、陈述权、辩护辩论权等的制度保障，落实罪刑法定、疑罪从无、非法证据排除等法律原则的法律制度。  


# 1.内涵

全民守法是指所有社会成员普遍尊重和信仰法律、依法行使权利和履行义务的状态。全民守法要求依法行使权利，依法履行义务，意味着依法维护自己的正当权益。  


# 2.推进全民守法

推进全民守法要着力增强全民法治观念，坚持把全民普法和守法作为依法治国的长期基础性工作；调动人民投身依法治国的积极性和主动性；不断加强公民道德建设，弘扬中华优秀传统文化

# 民事法律关系的主体

民事主体是指在民事法律关系中享有民事权利、承担民事义务的自然人、法人和非法人组织。  


# 1.民事权利能力

民事主体依法享有民事权利和承担民事义务的资格。  

自然人从出生时起到死亡时止，具有民事权利能力，其民事权利能力一律平等。

# 三、民事法律关系的客体与内容

1.民事法律关系的客体是指民事权利和义务所指向的对象。例如，所有权关系的客体是物，债权关系的客体是行为，知识产权关系的客体是智力成果和商业标记，人身关系的客体是人身利益。  

2.民事法律关系的内容是民事主体享有的权利和承担的义务。  

在民事法律关系中，权利与义务既是相互对立的，也是相互联系的。  


# 四、民法的基本原则

1.平等原则：民事主体在民事活动中的法  
律地位一律平等。2.自愿原则：按照自己的意思设立、变  
更、终止民事法律关系3.公平原则：合理确定各方的权利和义务。4.诚信原则：民事主体从事民事活动，秉  

持诚实，恪守承诺。  

5.守法和不违背公序良俗原则：民事主体从民事活动，不得违反法律，不得违背公序良俗。6.绿色原则：民事体从事民事活动应当有利于节约资源、保护生态环境。

# 五、人格权的保护与救济

1.侵权人承担民事责任。被侵权人可请求侵权人停止侵害、排除妨碍、消除危险、消除影响、恢复名誉、赔礼道歉。  


# 2.损害赔偿

（1）人身损害赔偿：医疗费、护理费、交通费、营养费、住院伙食补助费；因误工减少的收入；辅助器具费和残疾赔偿金；丧葬费和死亡赔偿金。  

（2）财产损失赔偿：被侵权人受到的损失或侵权人获得的利益  

（3）精神损害赔偿：造成严重精神损害，或因故意或者重大过失侵害自然人具有人身意义的特定物造成严重精神损害可进行赔偿  



# 、所有权

1.所有权是权利人对自己的不动产或者动产，依法享有的占有、使用、收益和处分的权利。依据主体的不同，我国将所有权划分为国家所有（全民所有）、集体所有、私人所有三类，均受法律平等保护。  


# 2.所有权的取得方式

不动产物权的设立、变更、转让和消灭，应当依照法律规定登记  

动产物权的设立和转让，应当依照法律规定交付。  

机动车等特殊动产的物权的设立、变更、转让和消灭，未经登记，不得对抗善意第三人。  


# 3.所有权的变更

一般物权变动=有权处分+有效合同+登记/交付  

特殊物权变动=无权处分+有效合同+登记/交付=善意取得  

4.共有  

不动产或者动产可以由两个以上组织、个人共有。共有包括按份共有和共同共有。

实施主要包括：制造、使用、销售、许诺销售、进口等。

发明专利的保护期为20年，实用新型专利的保护期为10年，外观设计专利的保护期为15年，均自申请日起计算。

商业秘密不需要公开，只要保密得当，可一直受到保护。但其他人独立作出相同的发明  

或以正当合法的方式获得该发明的技术信息，则可以实施发明，不构成侵权。  


# 1.商标的注册

商标是经营者用来区分商品和服务来源的标志。  

任何能够将商品与他人的商品区别开的标志，包括文字、图形、字母、数字、三维标志、颜色组合和声音等，以及上述要素的组合，均可以作为商标申请注册。  

我国的国家名称、国歌、国徽、国旗、军歌、军徽、军旗、勋章等相同或相近的标志，有害于社会主义道德风尚或者有其他不良影响的标志，县级以上行政区划的地名或公众知晓的外国地名，一般不得注册商标。  


# 2.商标的侵权与保护

（1）商标侵权①未经权利人许可，在同一种商品中使用相同商标；②在同一种商品中使用近似商标；③在类似商品中使用相同或者近似商标；④销售侵犯注册商标专有权的商品  


# （2）商标保护期

注册商标有效期为10年，到期注册人可申请续展，续展注册的有效期也是10年，续展次数不受限制。注册人只要依法续展，注册商标就可以长久地获得法律保护  


# 1.分类

（1）一般过错责任：行为人因过错侵害他人民事权益造成损害的，应当承担侵权责任。（2）过错推定：依照法律规定推定行为人有过错，其不能证明自己没有过错的，应当承担侵权责任。  


# 2.要件

侵害行为、损害后果、因果关系、主观过错（故意+过失）。  


# 无过错责任

行为人造成他人民事权益损害，不论行为人有无过错，法律规定应当承担侵权责任的，依照其规定。  


# 1.分类

（1）相对无过错责任：有免责事由的可少承担或不承担责任  

常见免责事由：与有过失（促成过失、过  

失相抵或者受害人过错），受害人敌意，不可抗力。  

常见无过错责任：用人单位工作人员执行公务造成他人损害；环境污染破坏。  

（2）绝对无过错责任：不存在任何免责事由。  

绝对无过错责任只有禁止饲养的烈性犬等危险动物造成他人损害  


# 2.要件

侵害行为、损害后果、因果关系  


# 三、诉讼时效

向人民法院请求保护民事权利的诉讼时效期间为三年。法律另有规定的除外。  

诉讼时效期间自权利人知道或者应当知道权利受到损害以及义务人之日起计算。法律另有规定的，依照其规定。  

但是，自权利受到损害之日起超过二十年的，人民法院不予保护，有特殊情况的，人民法院可以根据权利人的申请决定延长。  


# 考点6 合同与违约责任


合同是民事主体之间设立、变更、终止民事法律关系的协议。常见的合同有买卖合同、赠与合同、租赁合同、运输合同等。依法成立的合同，受法律保护。  


# 1.意思表示与合同形式

订立合同需要双方就某种民事法律关系的设立、变更、终止形成一致的意思表示。  

意思表示可以采用书面形式、口头形式或者其他形式；也可以明示或者默示作出意思表示。  


# 2.合同订立的过程

（1）要约  

要约是希望与他人订立合同的意思表示，该意思表示应当符合下列条件：  

①内容具体确定。必备条款包括当事人、标的、数量。  

②表明经受要约人承诺，要约人即受该意思表示约束。即有订立合同，产生私法效果的意思。  

要约发布、知悉、到达则生效  


# （2）承诺

承诺是受要约人同意要约的意思表示承诺应当在要约确定的期限内到达要约人。  

承诺的内容应当与要约的内容一致。受要约人对要约的内容作出实质性变更的，为新要约。  

实质性变更，即变更合同的：当事人的姓名或者名称和住所；标的；数量；质量；价款或者报酬；履行期限、地点和方式；违约责任；解决争议的方法。  


# 1.合同的成立与生效

合同成立意味着合同订立的意思表示已经完成，客观上存在民事法律行为的事实。  

合同生效即已成立的合同符合生效的法律  

要件，产生了当事人所追求的法律效果  

合同未成立不存在违约责任，可能存在缔约过失责任。  


# 2.合同的效力

（1）成立即生效。依法成立的合同，自成立时生效。  

有效的合同要求：①行为人具有相应的民事行为能力；②意思表示真实；③不违反法律、行政法规的强制性规定，不违背公序良俗。  

（2）附条件的合同，自条件成就时生效。附期限的合同，自期限届至时生效。  

（3）合同效力有瑕疵  

①效力待定。  

限制民事行为能力人签订的合同，经法定 代理人同意或者追认后有效  

签订时效力待定，同意追认后有效，拒绝追认或否认则无效。  

②合同可撤销。  

一方以欺诈手段签订的合同，受欺诈方有权请求人民法院或者仲裁机构撤销。  

一方或者第三人以胁迫手段，使对方在违背真实意思的情况下签订的合同，受胁迫方有权请求人民法院或者仲裁机构撤销  

基于重大误解签订的合同，行为人有权请求人民法院或者仲裁机构予以撤销。  

一方利用对方处于危困状态、缺乏判断能力等情形，致使合同成立时显失公平的，受损害方有权请求人民法院或者仲裁机构予以撤销。  

③合同无效  

无民事行为能力人签订的合同无效  

签订合同做出的是虚假的意思表示的合同无效。  

违反法律、行政法规的强制性规定的合同无效。  

恶意串通，损害他人合法权益的合同无效。  

以合法形式掩盖非法目的，因合法的形式是虚假的意思表示，合同无效。  


# 1.合同履行

当事人应当按照约定全面履行自己的义务。应当遵循诚信原则，根据合同的性质、目的和交易习惯履行通知、协助、保密等义务。  

全面履行、正确履行、适当履行，诚实信用，协作履行。  


# 2.合同变更

合同订立后双方协商一致可以变更合同  

除法律规定和当事人另有约定外，合同一方不能单方面解除合同  


# 1.违约的形式

预期违约：履行期届满之前，明确表示或以自己的行为表示不履行。  

实际违约：拒绝履行，履行不能，延迟履行，部分履行，瑕疵履行。  


# 2.违约责任的承担

继续履行、采取补救措施、损害赔偿金（法定）、违约金（约定）、定金罚则（约定，2倍）。  


# 3.免责事由（不可抗力）

不可抗力是不能预见、不能避免且不能克服的客观情况。当事人一方因不可抗力不能履行合同的，根据不可抗力的影响，部分或者全部免除责任。  


# 1.婚姻制度基本原则

婚姻自由，一夫一妻，男女平等，保护妇女、未成年人、老年人、残疾人的合法权益。禁止包办、买卖等干涉婚姻自由的行为，禁止重婚，禁止家庭暴力、家庭成员间的虐待和遗弃。  


# 2.结婚的要件

（1）实质要件  

①男女双方完全自愿。②男不得早于二十二周岁，女不得早于二十周岁。③符合一夫一妻制。④直系血亲或者三代以内的旁系血亲禁止结婚  


# （2）形式要件（依法登记）

依法登记取得结婚证，开始确立夫妻关系。未办理登记的需要补办登记，效力从双方均符合实质要件时算起。复婚需要重新登记未登记以夫妻名义生活，是同居关系。  

（3）婚姻效力  

①无效婚姻：重婚；有禁止结婚的亲属关系；未到法定婚龄②可撤销婚姻：胁迫婚，被胁迫方可请求撤销；一方婚前患有重大疾病，且未如实告知另一方，另一方可以请求撤销。  


# 1.人身关系

夫妻双方地位平等。人格独立，有使用自己姓名的权利。有参加生产、工作、学习、社会活动的自由，不受另一方干涉。平等享有对未成年子女的抚养、教育和保护的权利与义务。双方有相互扶养的义务。  


# 2.财产关系

（1）约定财产关系  

双方可以约定婚姻关系存续期间的财产范围，应采用书面形式，对双方均具有法律约束力。约定只对双方有效，若涉及善意相对人的债权债务，第三人必须知情  

（2）法定财产关系  

①法定共同财产  

工资、奖金、劳务报酬；生产、经营、投资的收益；知识产权的收益；继承或者受赠的财产，但明确只归一方的财产除外；其他应当归共同所有的财产  

②法定个人财产  

一方的婚前财产；一方因受到人身损害获得的赔偿或者补偿；遗或者赠与合同中确定只归一方的财产；一方专用的生活用品；其他应当归一方的财产。  

（3）夫妻双方的平等家事代理权  

双方在处理夫妻共同财产上权利平等，因日常生活需要处理共同财产的，任何一方均有决定权。对于双方的限制不得对抗善意第三人。  

（4）夫妻双方的债务关系  

①共同债务：共同签名、一方事后追认等共同意思表示，以个人名义为家庭日常生活需要所负债务。  

②个人债务：个人名义超出家庭日常生活需要所负债务。  


# 1.协议离婚

自愿离婚，签订书面协议，就子女抚养和财产分割达成一致。  

婚姻登记机关收到离婚登记申请起三十天内为离婚冷静期，冷静期期间任何一方可撤回。如若无撤回，三十天届满后的三十天内，双方亲自到婚姻登记机关申领离婚证。  


# 2.裁判离婚

一方要求离婚，先进行调解，调解无效应准予离婚。  

女方怀孕期间和分娩后一年内或终止妊娠六个月内，男方不得提出离婚  


# 3.离婚后的子女抚养与财产分割

离婚后，父母与子女的关系不因离婚消  

除。不直接抚养子女的一方有探望的权利，另一方有协助的义务。  

财产债务协议处理，协议不成人民法院判决。共同债务应当共同偿还。  


# 1.父母与子女

父母对未成年子女负有抚养、教育和保护的义务。成年子女对父母负有养、扶助和保护的义务。  

父母是未成年子女的监护人，职责是代理被监护人实施民事法律行为，保护被监护人的人身权利、财产权利以及其他合法权益等  


# 2.意定监护制度

具有完全民事行为能力的成年人，可以与其近亲属、其他愿意担任监护人的个人或者组织事先协商，以书面形式确定自己的监护人，在自己丧失或者部分丧失民事行为能力时，由该监护人履行监护职责  


# 1.遗产与继承权

（1）遗产  

继承从被继承人死亡时开始。遗产是自然人死亡时遗留的个人合法财产。  


# （2）继承权

继承人如果放弃继承，要在遗产处理前以书面形式做出意思表示，否则，视为接受  

受遗赠人在应当知道受遗赠之后六十日内，做出意思表示，没有表示的，视为放弃。  

以下行为之一，丧失继承权：杀害被继承人和其他继承人，遗弃、虐待被继承人，伪造、篡改、隐匿或销毁遗，欺诈、胁迫手段迫使、妨碍被继承人设立、变更或撤回遗。  


# 2.遗继承

（1）继承的法律效力：遗赠抚养协议>遗（遗赠）>法定继承  

（2）遗形式  

自书遗瞩需要亲笔书写、签名；公证遗瞩需要依法经公证机构办理  

代书遗、打印遗、录音录像形式的遗瞩，口头遗瞩需要有两个以上的无利害关系人在现场见证。  

遗可以撤回、变更，有数份遗且内容相抵触的，以最后的遗嘱为准。  


# （3）遗赠扶养协议

可与继承人以外的组织或个人签订遗赠扶养协议。  


# 3.法定继承

（1）法定继承顺序  

第一顺序：配偶、子女、父母；  

第二顺序：兄弟姐妹、祖父母、外祖父母。  

继承开始后，由第一顺序继承人继承，第二顺序继承人不继承；没有第一顺序继承人继承的，由第二顺序继承人继承。  

丧偶儿媳对公婆，丧偶女婿对岳父母，尽了主要养义务的，作为第一顺序继承人。  


# （2）代位继承

孙子女、外孙子女代位父母继承  

侄女、侄子，外甥女、外甥代位兄弟姐妹继承。  


# 4.遗产分割原则

一般均等分割；特殊困难又缺乏劳动能力的、尽了主要扶养义务或者与被继承人共同生活的可适当多分；不尽扶养义务的少分或者不分。  


# 5.债务的继承

税款和债务也需要继承，以遗产实际价值为限清偿。  

放弃继承的可以不负债务清偿责任  

债务的清偿法定优先，超过法定部分由遗嘱继承人和受遗赠人按比例清偿。  


# 劳动法调整对象

劳动法主要调整劳动者与用人单位在劳动过程中发生的社会关系，即劳动关系。  

禁止招用未满十六周岁的未成年人。  


# 劳动法基本原则

1.保护劳动者合法权益  
2.劳动者平等竞争。  
3.特殊劳动保护  


# 1.劳动合同的签订

建立劳动关系，应当订立书面劳动合同  

已建立劳动关系，未同时订立书面劳动合同的，应当自用工之日起一个月内订立书面劳动合同。用人单位与劳动者在用工前订立劳动合同的，劳动关系自用工之日起建立。  


# 2.劳动合同的内容

劳动者与用人单位之间的权利与义务。  

必备条款：劳动合同期限，工作内容和工作地点，工作时间和休息休假，劳动报酬，社会保险，劳动保护，劳动条件和职业危害防护，等等。  


# 3.劳动合同的签订原则

合法、公平、平等自愿、协商一致、诚实信用。  


# 4.劳动合同效力

劳动合同符合法律规定的条件才产生法律效力。具有下列情形之一的劳动合同无效或者部分无效：（1）欺诈、胁迫违背真实意思订立变更劳动合同。（2）用人单位免除自己法定责任，排除劳动者权利。（3）违反法律、行政法规强制性规定  


# 5.劳动合同解除

（1）用人单位与劳动者协商一致可解除劳动合同。（2）劳动者可单方解除劳动合同，包括提前预告解除；用人单位违法违约或危及劳动者人身自由和人身安全的情况下，劳动者随时、即时解除劳动合同  

（3）用人单位可单方解除劳动合同，包括劳动者存在过错被开除，劳动者没有过错但因不适应岗位需求或客观条件变化导致无法履行合同，以及经济性裁员。  


# 1.权利

取得劳动报酬的权利；休息休假的权利；获得劳动安全卫生保护的权利；享有社会保险和福利的权利；平等就业和选择职业的权利  


# 2.义务

完成劳动任务，提高职业技能，执行劳动安全卫生规程，遵守劳动纪律职业道德  


# 1.和解

发生纠纷后可当事人协商解决，协商一致后达成和解协议。和解协议无必须履行的法律  

效力，协商也不是处理劳动争议的必经程序  


# 2.调解

劳动争议调解委员会调解劳动争议。调解需在申请之日起30日内结束。调解协议书无必须履行的法律效力，调解也不是解决劳动争议的必经程序。  


# 3.仲裁

劳动仲裁是由劳动争议仲裁委员会以第三方的身份居中进行裁决的活动。如果当事人不愿意协商或调解、协商或调解不成、当事人达成和解或调解协议后不履行，都可以从当事人知道或者应当知道其权利被侵害之日起一年内向劳动争议仲裁委员会提出书面仲裁申请，由仲裁委员会依法作出具有法律约束力的裁决。根据劳动法律、法规，除特定情形外，未经劳动仲裁程序，当事人不得直接向人民法院提起诉讼。  


# 4.诉讼

除一裁终局以外的劳动争议案件的仲裁裁决，当事人不服可自收到裁决书十五日内向人民法院提起诉讼。  

考点9 就业、创业与消费者权益保护  


# 1.非法人组织

个人独资企业、合伙企业组织结构相对简单，出资人以个人财产对企业债务承担无限责任。  


# 2.法人

主要形式：有限责任公司、股份有限公司。  

公司指资本由股东出资构成，股东以其认缴的出资额或者认购的股份为限对公司承担责任，公司以其全部资产对公司债务承担责任，并依法设立的企业法人。  

公司具有独立人格：财产独立，名义独立，责任独立。  

分公司是总公司下属的直接从事业务经营活动的分支机构或附属机构。分公司不具有企业法人资格，不独立承担民事责任。  

子公司是与母公司相对应的法律概念  

子公司受母公司的实际控制，具有法人资格，可以独立承担民事责任。  

有限责任公司股本不拆分、股东（股东人数上限为50人）按出资比例行使表决权，资产流动性弱，规模相对较小，资本无需划分为等额股份。  

股份有限公司股本等额拆分，一股一权，股东以其所认购的股份，享受权利、承担义务。公司向社会发行股票融资，资产流动性强。规模相对较大。  


# 3.组织结构

（1）股东大会与董事会：权力机关，决定  

公司的投资计划、方案，经营方针计划，以及相关人事任免。  

（2）总经理：执行机关，组织实施投资计划和经营计划。  

（3）监事会：监督机关，监督公司财务、事项和工作。

# 2.诚信经营

保障消费者安全消费的权利；保障消费者知情权；尊重消费者自主选择权；遵循自愿、公平、平等、诚信的原则

2.企业所得税：针对企业经营所得征税个人独资企业与合伙企业征收个人所得税

# 1.主体：主权国家和国际组织

（1）主权国家

# 国际组织概论

若于国家的政府或民间团体经协商联合建立的经常性组织  


# 1.分类

成员性质：政府间国际组织、非政府间国际组织  

地理范围：世界性国际组织、区域性国际组织。  

职能范围：一般性国际组织、专门性国际组织。  


# 2.机构

议事和决策机构：一般来说，全体成员  

参加的大会是议事和决策机构，享有广泛的权力。  

执行机构：大会休会期间，部分成员组成执行机构。  

秘书处：负责处理日常事务，为其他机构提供服务。  

司法机构：有些国际组织还设有司法机构，解决成员间的矛盾和纠纷。  


# 3.特征

国际性。国际组织是国家、地区之间的组织，两个以上国家、地区的政府、团体或个人才能建立国际组织  

目的性。国家组织有特定的宗旨。宗旨是其开展活动的指导原则  

组织性。国际组织设有多层次的机构来开展活动。  

自主性。国际组织代表各成员的共同利益而非某个特定成员的单方利益。在一定程度上，国际组织自主开展活动，独立运作，其决议具有某种强制力。  


# 4.产生及其作用

（1）产生  

国际组织是国际关系发展到一定历史阶段的产物，其形成和发展反映了生产力发展的水平和要求，也表明人类交往的日益频繁、面临的问题日益复杂。  


# （2）作用

国际组织可以促进国际社会在各个领域开展交流、协调、合作，调停和解决国际政治冲突、经济纠纷，促进世界和平与发展。  

国际组织参与国际事务受多种因素影响，存在局限性，个别大国将国际组织作为推行霸权主义、强权政治的工具

# 2.运动

（1）运动与物质的辩证关系  

物质是运动的物质，运动是物质固有的根本属性和存在方式。离开运动谈物质会陷人形而上学。  

运动是物质的运动，物质是运动的承担者。离开物质谈运动会陷入唯心主义。  

（2）运动与静止的辩证关系  

运动是无条件的、永恒的、绝对的。静止是有条件的、暂时的、相对的。绝对运动与相对静止是统一的。  


# 3.规律

（1）原理：规律是事物运动过程中固有的本质的、必然的、稳定的联系。  

①规律是客观的，不以人的意志为转移。人可以认识和利用规律②规律是普遍的。自然界、人类社会和人的思维都遵循其固有的规律。  

（2）方法论  

①规律客观性、普遍性原理要求我们尊重 客观规律，按客观规律办事。违背规律会受到 规律的惩罚。  

②规律是可以认识和利用的。人在规律面前不是无能为力的，人可以在认识和把握规律的基础上，根据规律发生作用的条件和形式，利用规律改造客观世界，造福人类。

# 3.不平衡性

（1）主次矛盾  

①处于支配地位，对事物发展方向起决定作用的矛盾就是主要矛盾。②处于从属地位，对事物发展不起决定作用的矛盾则是次要矛盾③主要矛盾和次要矛盾相互影响、相互制约，并在一定条件下相互转化方法论：把握重点，集中力量解决主要矛盾；统筹兼顾，妥善处理次要矛盾。  

（2）矛盾主次方面  

①处于支配地位，决定事物性质的一方是矛盾的主要方面②处于被支配地位的方面是矛盾的次要方面。③矛盾的主要方面与次要方面相互排斥又相互依赖，并在一定条件下相互转化  

方法论：看问题要全面统筹，同时也要分清楚主流和支流  

（3）两点论与重点论  

①坚持两点论，既要看到主要矛盾，又要看到次要矛盾；既要看到矛盾的主要方面，又要看到矛盾的次要方面②坚持重点论，要着重把握主要矛盾，要抓重点；要着重把握矛盾的主要方面，要抓主流。③两点论是有重点的两点，不是均衡论。重点论是两点中的重点，而不是一点论。

# 1.质、量、度

质：一事物成为自身并区别于其他事物的内部固有的规定性。  

量：事物存在和发展的规模、速度、程度等用数量表示的规定性，以及事物构成因素在空间上的排列组合方式  

度：一事物保持自身质的稳定性的量的限度。  

关节点／临界点：度的两端，事物量变达到的极限，引起质变的节点。  


# 2.量变、质变及其辩证关系

（1）定义  

量变：事物量的变化，表现为事物变化发展过程中连续性和渐进性。  

质变：事物质的变化，表现为事物变化发展过程中飞跃性和间断性。  

区分标志：事物的变化是否超过度的范围。  

（2）辩证关系  

①量变是质变的必要前提。渐进性与连续性的积累产生了飞跃性与间断性。②质变是量变的必然结果。飞跃性与间断性包含着渐进性与连续性。  


# 3.方法论

（1）保持特定的质，就需要控制在度的范围内，防止过犹不及。  

（2）不需要事物特定的质，积极创造条  


# 件，促进事物质的飞跃

（3）适度原则即在“度”的范围内寻求和把握事物发展的适当程度，积极认识和推动事物发展达到最佳状态  


# 1.肯定与否定

肯定：事物保持自身存在的方面，处于优势时事物会保持其原有的性质和自身的存在。  

否定：促使事物灭亡的方面，在事物发展中取得支配地位时事物会改变自己的根本性质，实现对自身的否定。  


# 2.辩证否定观

肯定一否定一否定之否定  

（1）辩证否定是自我否定，通过自我否定实现更新（2）辩证否定的实质是“扬弃”，即取其精华，去其糟粕，否定不是全盘否定，而是包含肯定。（3）辩证否定实现新旧事物的联系，实现事物的发展事物发展的过程是肯定一否定一否定之否定的过程。  

否定之否定是对矛盾统一体的否定，这个过程保留了旧统一体中合理的因素、改造了其中不合理的因素，实现了事物的发展和更新。  


# 1.从感性具体到思维抽象

（1）定义  

感性具体：通过感官感知到的事物的现象和外部联系  

思维抽象：从多样性统一的事物整体中抽取某一方面的本质规定，或者从个性中抽取共性的思维活动。  

（2）方法  

①分离：起始环节、不考虑研究对象与其他对象的联系。②提纯：关键环节、排除干扰因素，研究某一方面的性质和规律③简略化：表述思维抽象结果④理想化：在思维中对认识对象存在状态的纯然构想  


# 2.从思维抽象到思维具体

（1）思维具体：在思维中把事物各个方面的本质规定按照其内在联系综合起来，形成关于事物整体的本质和规律的认识（2）方法：上升的起点往往是某一认识对象的矛盾焦点。

创新思维  



# 1.含义

人们在实践中破除迷信、超越常规、有所发现、有所发明的思维活动。  

表现为思路新，方法新，结果新  

创新思维以实践为基础，也离不开对前人已有成果的继承。既要运用逻辑思维活动和常规思维方法，也要运用非逻辑思维活动、非常规的思维方法  


# 2.特征

（1）思路具有多向性，从多个不同角度和方向思考问题（2）步骤具有跨越性，对推理步骤的省略或跨越。（3）结果具有独特性，用独特巧妙的方法解决问题  


# 1.含义

（1）事物之间是普遍联系的，联想思维是对事物之间普遍联系的反映（2）联想思维将对看似不相关对象的认识联系在一起，具有跨越的联结性（3）联想思维具有非逻辑制约的畅想性，包含着构想、想象甚至幻想  

衡量联想水平的标准：“联结”的速度和“畅想”得到的结果的数量。  


# 2.方法

（1）迁移是联系思维的“联结”方式  

将不同认识对象的性质、作用等进行位置变迁与功能移植。  

迁移为创新思维开拓可能的思路，为创新思维搭建由此及彼的桥梁  

（2）想象是联想思维的“畅想”方式  

在头脑中对已有的事物表象进行加工、改造，通过重新组合而产生新的事物形象  

把抽象的思想、概念形象化，按照有意想象的要求去思考，是一种形象化的想象。  

通过提出“假如”式的问题，将与事实相反的情况作为事物发展的一种条件，仿照事物之间的条件关系，推测事物发展的可能前景。  

想象可以帮助人们明确创新思维目标。

# 2.认识的发生

实践与认识的辩证关系原理  

（1）原理  

①实践决定认识。实践是认识的来源，实践是认识发展的动力（认识产生于实践需求，实践为认识提供工具，实践锻炼人的认识能力），实践是检验认识的真理性的唯一标准（实践是连接主客观的桥梁，真理只有主客观结合才能检验；实践具有直接现实性，把观念转变为现实存在加以对照），实践是认识的目的和归宿。  

②认识对实践具有反作用。真理、科学理论对实践具有促进作用，谬误阻碍实践发展。  

（2）方法论：我们首先要坚持实践第一性，积极投身实践，在实践中检验和发展认识；还要重视认识的反作用，特别是科学理论对实践的巨大指导作用，坚持理论和实践相结合的原则，做到理论和实践的具体的历史的统一。

感性认识是认识的初级阶段，有待于 发展、深化为理性认识。  

理性认识是认识的高级阶段，依赖于感性 认识。

# 社会发展规律

1.社会生活本质上是实践的，劳动是社会历史的起点。2.社会存在决定社会意识。社会意识具有相对独立性。  

社会意识对社会存在具有反作用。落后的社会意识对社会的发展起阻碍作用，先进的社会意识可以正确地预见社会发展的方向和趋势，对社会发展起积极的推动作用。  

3.生产力决定生产关系，生产力的状况决定生产关系的性质，生产力的变化发展，迟早会引起生产关系的变革。生产关系对生产力具有反作用。当生产关系适合生产力发展状况时，它对生产力的发展起推动作用；当生产关系不适合生产力发展状况时，它对生产力的发展起阻碍作用。  

4.经济基础决定上层建筑，上层建筑对经济基础具有反作用。当上层建筑适合经济基础的状况时，它促进经济基础的巩固和完善；当它不适合经济基础状况时，会阻碍经济基础的发展和变革。  


# 社会发展的总趋势

1.社会历史发展的总趋势是前进的、上升的，发展的过程是曲折的。社会发展是在生产力和生产关系、经济基础和上层建筑的矛盾运动中，在社会基本矛盾的不断解决中实现的。  

2.在阶级社会里，社会基本矛盾的解决主要是通过革命实现的，阶级斗争是推动阶级社会发展的直接动力。社会主义社会的基本矛盾是非对抗性的矛盾，要通过改革解决  

3.改革是社会主义制度的自我完善和发展，是社会主义社会发展的直接动力。其根本目的是使生产关系适应生产力的发展，使上层建筑适应经济基础的发展

# 1.思维的含义

思维有广义和狭义之分。广义的思维与意识同义，狭义的思维与理性认识同义。  

感性认识是人脑对客观事物的现象和外部联系的反映，是认识的初级阶段。理性认识指认识的高级阶段，是对事物的本质及其规律的反映  


# 2.思维的特征

（1）间接性：思维能透过事物的现象，揭示事物的本质和规律，实现对未知事物的认识（2）概括性：思维能够从多种事物及其各种各样的属性中抓住内在的、共同的、本质的属性（3）能动性：思维能够提炼加工感性材料，形成有别于客观实际的认识  


# 1.分类

（1）从思维的方向看，分为向不同方向扩散的发散思维和向同一方向收敛的聚合思维。（2）从思维对认识对象的思考角度看，分为整体地认识对象的综合思维和分别地认识对象的分析思维（3）从思维反映认识对象的方式看，分为用联系、发展、全面的观点看待事物和思考问题的辩证思维和用孤立、片面、静止的观点看待事物和思考问题的形而上学思维  


# 2.思维基本形态的特征

根据思维运行的基本单元不同，分为抽象思维和形象思维，是思维的基本形态  

（1）抽象思维  

①认识对象为概念、判断、推理等，揭示事物的本质和规律②主要特征：基本单元的概念性、运行方式的推导性、思维表达的严谨性  


# （2）形象思维

①运用联想、想象和幻想等反映认识对象，触及事物的本质和规律②主要特征：基本单元的形象性、运行方式的想象性、思维表达的情感性。  

（3）二者各有其功能和作用，但又相辅相成  


# 1.逻辑与逻辑学

逻辑学是用来研究思维形式中的规律与规  

则的学科，是一个工具性学科。  

（1）狭义逻辑学即形式逻辑，用来研究思维形式结构及其规律  

形式逻辑特别关注推理问题，即由前提推导结论。论证是用论点证明论据，用这种方式去说服或者反驳他人观点，构成论辩。  

（2）广义逻辑学包含狭义逻辑学，还包括研究辩证思维中的规律、规则与方法的辩证逻辑。  


# 2.逻辑思维的基本要求

（1）同一律：思维确定性的要求  

①公式：A是A。  

②同一思维过程中，每一思想必须保持自身同一性。不能混淆概念，不能转移论题③它不否认认识对象的复杂多样性及其变化和发展。  

（2）矛盾律：思维一致性的要求。  

①公式：A不是非A。  

②在同一时间、从同一方面、对同一对象所形成的论断“A”和它的否定论断“非A”不能同时为真，其中必有一假。  

（3）排中律：思维明确性的要求。  

①公式：A或者非A。  

②在同一时间、从同一方面、对同一对象所形成的论断“A”和它的否定论断非“A”不能同时为假，其中必有一真

# 1.判断

（1）含义  

判断就是对认识对象有所断定的思维形式。  

判断是通过语句表达的，判断是语句的思想内容，语句是判断的语言形式。  

（2）特征  

①对认识对象有所断定，断定的方式包括肯定、否定。  

②有真假之分，其区分标准是断定认识对象是否符合实际  

判断来源于人们的社会实践，其正确与否要经受社会实践的检验。  


# （3）类型

以判断本身是否包含其他判断为标准，分为简单判断和复合判断

①推理规则  

规则一：有且只有三个不同的项，否则会犯“四概念”的错误。  

规则二：三段论的中项，至少要周延一次，否则中项不周延  

规则三：前提中不周延的项，结论中不得周延，否则大项／小项不当扩大。  

规则四：两个否定前提不能推出结论；当且仅当前提中有一否定，结论才否定。  

规则五：两个特称前提不能推出结论；前提中有一特称，结论必须特称。  

②常见二次结论  

结论一：前提：所有A是B，所有B是C。  
结论：所有A是C。  
结论二：前提：有的A是B，所有B是C。  
结论：有的A是C。

# 3.假言判断

（1）含义：某种情况的存在（不存在）是另一种情况存在（不存在）的条件。（2）格式：联结项+前件+联结项+后件。  

（3）常见表达  

①S→P：如果S那么P，只要S就P，S必须P。②S→P：只有P才S。③非S→P，非P→S：除非S，否则P。DS←→P：S当且仅当P。  

（4）真假推理  

①充分条件：S→P，有S一定有P，无S未必无P，S真P真②必要条件：S→P，无P一定无S，有P未必有S，P假S假。③充要条件：S←→P，有S一定有P，无S一定无P。④S←→P=非P→非S=非S或P=非（S且非P)。  


# 1.含义

从个别、特殊的前提推导出一般性的结论。  

推理的前提不涉及全部的对象，是不完全归纳推理，前提结论不保真，是或然推理  

推理的前提涉及全部的对象，是完全归纳推理，前提结论保真，是必然推理  


# 2.归纳推理的方法

（1）完全归纳推理真实可靠的条件①断定个别对象情况的每个前提都是真实的。②所涉及的认识对象，一个都不能遗漏  

（2）不完全归纳推理如何提高可靠度：寻找因果联系。  

因果联系：事物或现象之间的引起与被引起的关系。  


# ①求同法

被考察的现象a出现在多个场合，且在这些场合中，只有一个有关因素A是共同的。则A与a有因果联系。  


# ②求异法

被考察的现象a出现在第一场合，没有出现在第二场合。第一场合有A因素，第二场合没有A因素。其他因素均相同，则A与a有因果联系。  


# ③共变法

被考察的现象a发生变化，且有一个因素A也随之发生变化，则A与a有因果联系。  


# 1.含义

根据两个或两类对象在一些属性上相同或相似，推出他们在其他属性上也相同或相似。  

类比不同于比较，也不同于比喻。类比推理是或然推理  


# 2.提高类比推理的可靠程度

（1）类比的根据越多越好。（2）相同属性越接近本质属性，相同属性与推出属性之间的相关程度越高越好（3）前提中确认的属性不应该有与结论相互排斥的属性
